# AutomataIA - Instalador Automático de IA para Desarrollo

Este script automatiza la instalación y configuración de herramientas de desarrollo con IA en español:

- **Codeium**: Autocompletado inteligente y gratuito
- **Tabby**: Asistente IA privada y local
- **Continue.dev**: Asistente IA integrado en VS Code
- **Ollama**: Ejecuta modelos de lenguaje en tu máquina local

## Características

✅ Instalación automática sin dependencias previas
✅ Detección e instalación de dependencias del sistema
✅ Soporte para Windows, macOS y Linux
✅ Configuración automática en español
✅ Actualizaciones automáticas de herramientas

## Requisitos prerequisitos

- Ninguno. El script detecta e instala todo lo necesario automáticamente.

## Uso

```powershell
python automatiaisetup.py
```

El script:
1. Detecta tu sistema operativo
2. Instala dependencias faltantes (Docker, Python, Curl, VS Code)
3. Instala Ollama, Tabby y Continue
4. Descarga e instala modelos en español
5. Configura todas las herramientas para su uso inmediato

¡Descubre una experiencia de desarrollo completamente optimizada sin instalar nada manualmente!
